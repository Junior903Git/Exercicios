Marcador de texto

tambem podemos marcar um texto assim como uma caneta marcadora de texto amarela

podemos tambem trocar a cor padrao do marcador por outra

dessa forma somente este mark tera essa coloracao diferente nenhum outro herdara essa configuracao, se caso quiser trocar a cor de todos mark que voce for criar deve ser feito de outra maneira como no exemplo abaixo

                
    <style>
        mark { background-color: blue; }
    </style>

use esse exemplo na parte head do html, assim todos marks do codigo mudara a cor
